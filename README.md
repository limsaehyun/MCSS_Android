MCSS(Minecraft Simple Searcher)는 쉽고 빠르게 마인크래프트 서버를 검색할 수 있는 앱입니다.
모바일 앱에서도 PE, PC Minecraft의 서버를 검색할 수 있습니다.
다른 서비스와 달리 MCSS는 무료이며 오픈 소스입니다.
